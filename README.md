# 📖 Knowledge Passbook (知识存折)

> **拒绝“虚假勤奋”，让每一份阅读都复利增值。**

**Knowledge Passbook** 是一款专注于**知识内化**与**资产化管理**的智能工具。它通过“教练式”AI 交互与“异步持久化”机制，打破“看书即忘”的魔咒，帮助用户将碎片化的信息孵化为系统性的个人知识资产。

---

## 💡 为什么需要 Knowledge Passbook？

在信息爆炸的时代，我们经常陷入**“虚假勤奋”**的陷阱：
*   **收藏夹吃灰**：看到好文章先收藏，却再也没有打开过。
*   **照搬式笔记**：机械地复制粘贴书本内容，大脑从未真正加工。
*   **知识孤岛**：学到的知识点零散破碎，无法形成解决问题的系统。

**Knowledge Passbook 的核心逻辑是：学习不是为了“存起来”，而是为了“长出来”。**

---

## 🛠️ 核心机制：从“信息输入”到“知识资产”

### 1. 五维内化法 (The 5-Question Framework)
当你完成一段学习后，系统会强制触发“内化孵化器”，要求你通过回答 5 个核心问题来完成知识的初次“显影”：
*   **定义**：这个知识点究竟在讲什么？
*   **解构**：你用自己的语言怎么理解它？
*   **场景**：它可以用在什么地方？
*   **实操**：具体如何使用？
*   **价值**：为什么它值得被存入你的“知识银行”？

### 2. 教练式 AI 审核 (AI Coaching & Feedback)
AI 不只是存储器，更是你的**私人知识教练**。它会检查你的回答：
*   **识别抄袭感**：检测你是直接照搬原句，还是已经转化成了“人话”。
*   **深度引导**：如果你表述模糊，AI 会给出修改建议，引导你思考得更深。
*   **确认存入**：只有当 AI 认可你已经完成内化，才会生成一张**“知识洞见卡”**并存入银行。

### 3. 时间线演变 (Insight Versioning)
知识是流动的。随着经验增长，你可能对旧知识有新见解。
*   **版本追踪**：你可以随时修改洞见卡，系统会记录你对同一知识点在不同时期的理解。
*   **成长轨迹**：打开“知识存折”，你可以清晰看到自己认知进阶的全过程。

### 4. 知识关联图谱 (Neural Knowledge Graph)
拒绝孤岛。系统会根据知识点间的逻辑关联自动生成**动态关系图**。
*   **自动织网**：让新旧知识自动“勾连”。
*   **系统化思考**：当你调取一个存折时，相关的知识脉络会一并浮现，助你实现方案的系统性输出。

---

## 🌟 项目亮点

*   **异步持久化机制**：不追求即时记忆，通过结构化复述实现深度长时记忆。
*   **孵化器理念**：将每一个知识点视作一颗种子，通过 AI 交互完成从“信息”到“洞见”的孵化。
*   **资产化视角**：像存钱一样积累知识，每一张“洞见卡”都是你的数字资产，可视化你的认知财富。

## 技术栈
- **前端**：Next.js 14, Tailwind CSS, Framer Motion
- **后端**：FastAPI, SQLAlchemy, Python 3
- **架构**：分离式前后端，可扩展到 Supabase/PostgreSQL。

## 快速开始

### 如何参与？

```bash
git clone https://github.com/zengruiting/knowledge-passbook.git
...
```

### 1. 后端启动
进入 `backend` 目录，安装依赖并启动服务：

```bash
cd backend
pip install -r requirements.txt
# 默认使用本地 SQLite 数据库，无需特殊配置即可直接运行
python -m uvicorn app.main:app --reload --port 8000
```

*(可选) 如需配置远程数据库等环境变量，可参考 `backend/.env.example`。*

### 2. 前端启动
进入 `frontend` 目录，安装依赖并启动开发服务器：

```bash
cd frontend
npm install
npm run dev
```

在浏览器中访问 [http://localhost:3000](http://localhost:3000) 即可使用。
**如果你不想自己部署，想快速使用，可直接访问 https://mashangfa.site**
